# Dispatch Protocol

The operational delegation layer for autonomous AI agents.

**Three commands. Your AI agents run themselves.**

```bash
dispatch discover --ssh root@my-vps
dispatch plan
dispatch deploy
```

See [VISION.md](VISION.md) for the full design document.

## Install

```bash
pip install dispatch-protocol
```

## Development

```bash
pip install -e ".[dev]"
dispatch --help
```
